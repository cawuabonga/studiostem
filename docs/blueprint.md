# **App Name**: CAP FAP Connect

## Core Features:

- Login Authentication: Implement login functionality with email/password and Google sign-in.
- Role-Based Access: Assign roles (student, teacher, coordinator, admin) to users upon registration.
- User Registration: Collect user information including profile picture during registration.
- Welcome Dashboard: Display a welcome dashboard with the user's profile picture, name, and role after login.
- Sidebar Navigation: Implement a side navigation menu on the dashboard for future menu options.
- Profile Display: Show user profile picture, name, and role in the side navigation menu.
- Announcements Carousel: Display an image carousel with institutional announcements/images on the login page, editable by the admin.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey formality and trustworthiness.
- Background color: Light desaturated blue (#D5DAE8), close to the primary hue, to create a professional and clean look.
- Accent color: Gold (#FFCE30,), analogous to the primary but distinct in brightness and saturation, for highlighting important elements.
- Headline font: 'Playfair', a modern sans-serif with an elegant feel; body font: 'PT Sans', a humanist sans-serif that is both modern and warm.
- The institution's logo should be placed at the top center of the login page. The image slider should be placed on one side of the login screen to display announcements and images.
- Use formal and professional icons throughout the application, in line with the institutional theme.
- Subtle transitions and animations for a polished and professional user experience. The image carousel should transition smoothly between images.
